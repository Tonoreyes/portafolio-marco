Mis Proyectos

Lista de tares

Propósito del proyecto: Esta aplicación fue diseñada para ayudar a los usuarios a gestionar sus tareas diarias de forma sencilla y visual. Permite agregar, editar, marcar como completadas y eliminar tareas, todo desde una interfaz amigable y responsiva.

Problema que resuelve: Muchas personas olvidan sus pendientes o no tienen una forma rápida de organizarlos. Esta app resuelve ese problema al ofrecer una herramienta accesible que guarda las tareas localmente en el navegador, sin necesidad de registro o conexión a internet.

Características clave:Persistencia de datos con localStorage.Edición en línea de tareas con validación.Interfaz visual clara con estilos personalizados en CSS.Interacción fluida mediante eventos como keypress y DOMContentLoaded.

Tecnologías: HTML, CSS, JavaScript

El clima de tu ciudad

Propósito del proyecto:Crear una aplicación web interactiva que permita consultar el clima actual de cualquier ciudad, integrando diseño y persistencia de datos

Problema que resuelve: Esta app ofrece una experiencia ligera y directa para consultar el clima. Permite guardar la última ciudad buscada para facilitar futuras consultas. Presenta la información de forma clara, y accesible desde cualquier navegador.

Características clave: Búsqueda por ciudad, Consulta a API de OpenWeather, Manejo de errores, Guarda la última ciudad para mostrarla al iniciar y Mensajes de error si la ciudad no se encuentra

Tecnologías: React, Vite, Libreria gh-pages y APIs
